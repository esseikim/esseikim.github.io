- 문제 풀이

```sql
SELECT DR_NAME,
       DR_ID,
       MCDP_CD,
       TO_CHAR(HIRE_YMD, 'YYYY-MM-DD') AS "HIRE_YMD"
FROM DOCTOR
WHERE MCDP_CD IN ('CS', 'GS')
ORDER BY HIRE_YMD DESC, DR_NAME ASC;
```

```sql
SELECT DR_NAME, 
       DR_ID, 
       MCDP_CD, 
       TO_CHAR(HIRE_YMD, 'YYYY-MM-DD') AS "HIRE_YMD"
FROM DOCTOR
WHERE MCDP_CD = 'CS' OR MCDP_CD = 'GS'
ORDER BY HIRE_YMD DESC, DR_NAME ASC;
```
