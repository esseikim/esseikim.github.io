- 문제 풀이

```java
class Solution {
    public String solution(String n_str) {
        int idx = 0;
        
        while (idx < n_str.length() && n_str.charAt(idx) == '0') { // while 루프를 통해 문자열의 왼쪽부터 0을 만날 때까지 인덱스를 이동하고,
            idx++;
        }

        return n_str.substring(idx); // 0이 아닌 숫자가 나오는 지점부터의 문자열을 반환한다.

    }
}
```
