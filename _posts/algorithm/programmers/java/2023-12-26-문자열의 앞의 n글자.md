- 문제 풀이

```java
class Solution {
    public String solution(String my_string, int n) {
        return my_string.substring(0, n); // 문자열을 자르는 substring 메서드, my_string의 0번째 위치부터 n번째 위치 직전까지의 문자열을 반환
    }
}
```
